# TimbermanASL
Auto Splitter for Timberman (PC)

- Starts when your score is 1
- Splits at selected scores (you can choose 100, 250, 500, 750, 1000, 1500, and 2000). 
- Resets when the score is 0

Tested on Timberman v2.10.
